BIDS Tool

“BIDS Tool is a building information design support system. It is useful for decision makers, architects, professors, and all people related to building industry, to learn the new technology and how it improves the building, occupants, as well as environments."

01. Background Research

Architects

Use BIDS tool as a library.

Managers

Use BIDS tool as a communication platform.

Clients

Use BIDS tool as a financial supporting tool.

Common goals:

Learn how to use the BIDS calculator.
Filter the studies based on building types and design strategies.
Calculate the benefit of certain type of technology for input building type.
Generate a exportable report for the result.
Access the short summary and the original article of the case study.
See ways to adopt useful technologies and products which should be used.

“The relationship between target users and BIDS Tool was illustrated through usability evaluation”

02. Contextual Inquiry

“The contextual inquiry result shows that the hidden links between pages and the unclear information of how to use the BIDS tool are two most frequent problems users encountered.”

03. Heuristic Evaluation

“Based on the low-fi prototype, we did the heuristic evaluation to improve the interface further.
More than 80% of the usability issues are solved.
Thestep to step filter and the clickable BIDS introduction are enhanced.”

04. BIDS Solution

Categorized home page make access clear

Step by step filter helps with searching process

Message box to improve communication

Easy to select and save the result

Unified platform for uploading and managing
